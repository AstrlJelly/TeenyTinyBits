# Teeny

Teeny is a minimalist ECS game engine, focused on optimization and simplicity. Written in C++, using OpenGL.

Data is laid out for cache efficiency and consistency; memory locations of components will never change unless intentionally changed.



## Contributing

Teeny is EXTREMELY work in progress. Pull requests without direct contact will not be accepted.

However, issues are fully welcome. Complain all you like. :\]
